# ERPNext Docker Image

All freaze ERPNext and all installed apps.

## Local

Build:

```bash
docker-compose build
```

Start:
```bash
docker-compose up --force-recreate -d
```

Open in Browser

[http://127.0.0.1:8000](http://127.0.0.1:8000)
